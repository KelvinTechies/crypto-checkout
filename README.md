# Crypto Checkout - Recipient Details Flow

A modern, production-ready crypto payment checkout experience built with Next.js 16, TypeScript, and Tailwind CSS. This project demonstrates a multi-step form flow for collecting bank and contact information with real-time validation.

## 🚀 Live Demo

- **Deployed URL**: [your-vercel-url.vercel.app]
- **Repository**: https://github.com/KelvinTechies/crypto-checkout


## ✨ Features Implemented

### Core Requirements ✅
- ✅ **Two fully responsive pages** - Bank Details & Contact Details screens
- ✅ **TypeScript** - Complete type safety with proper interfaces
- ✅ **Responsive Design** - Mobile-first approach, works on all screen sizes
- ✅ **Clean Component Structure** - Separated concerns, reusable components
- ✅ **Form State Management** - React hooks for efficient state handling
- ✅ **Accessibility** - ARIA labels, keyboard navigation, focus management

### Bonus Features ⭐
- ✅ **Real-time Form Validation** - Validates on blur with user-friendly error messages
- ✅ **Progressive Error Display** - Shows errors only after user interaction
- ✅ **Form Data Persistence** - State maintained when navigating between steps
- ✅ **Phone Number Auto-formatting** - Formats as user types (000 - 000 - 00000)
- ✅ **Custom Dropdown Components** - Fully accessible with keyboard navigation
- ✅ **Input Sanitization** - Account numbers restricted to 10 digits, phone numbers formatted
- ✅ **Loading States** - Smooth transitions between form steps
- ✅ **Error Recovery** - Clear error messages with guidance on how to fix issues
